# Flickers AI - Портативная версия

## 📦 Что это?

Портативная версия Flickers AI - это автономное приложение, которое не требует установки. Просто запустите EXE файл и начните работу!

## 🚀 Быстрый старт

### Вариант 1: Использование батника (рекомендуется)

1. Распакуйте архив в любую папку
2. Запустите `Запустить Flickers AI.bat`
3. Приложение автоматически запустит backend и откроет интерфейс

### Вариант 2: Ручной запуск

1. Откройте терминал в папке `backend`
2. Запустите: `python main.py`
3. Откройте папку `frontend/release/win-unpacked`
4. Запустите `Flickers AI.exe`

## ⚙️ Требования

- Windows 10/11 (64-bit)
- Python 3.8 или новее
- 4 GB RAM (рекомендуется 8 GB)
- 500 MB свободного места на диске

## 📝 Настройка

### Backend конфигурация

Отредактируйте файл `backend/.env`:

```env
# API ключи для AI моделей
ANTHROPIC_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here
GOOGLE_API_KEY=your_key_here

# Настройки сервера
HOST=127.0.0.1
PORT=8000
```

### Frontend конфигурация

Backend URL настраивается автоматически на `http://localhost:8000`

## 🔧 Устранение неполадок

### Backend не запускается

1. Проверьте, что Python установлен: `python --version`
2. Установите зависимости: `cd backend && pip install -r requirements.txt`
3. Проверьте, что порт 8000 свободен

### Приложение не открывается

1. Проверьте, что backend запущен (должен быть доступен http://localhost:8000)
2. Попробуйте запустить от имени администратора
3. Проверьте антивирус - добавьте приложение в исключения

### Ошибка "Python не найден"

Установите Python с официального сайта: https://www.python.org/downloads/

При установке обязательно отметьте "Add Python to PATH"
